
# Arcane Kitchen Color Palette
This the curated color palette for Arcane Kitchen, designed to evoke a mystical, alchemical atmosphere while ensuring usability and accessibility.

| Name | Hex | Use | Description |
|------|-----|-----|-------------|
| **Cauldron Green** | `#3A5A40` | Primary brand color, buttons, accents | Deep forest green — evokes herbs, potions, and alchemy; grounding and natural. |
| **Spellbook Brown** | `#5C4033` | Backgrounds, dividers, muted UI elements | Rich earthy brown — gives warmth, like a wooden workbench or old spellbook. |
| **Golden Ember** | `#D4A017` | Highlights, hover states, call-to-action buttons | Soft golden yellow — adds warmth and focus, like candlelight reflecting off brass. |
| **Arcane Violet** | `#6C4AB6` | Secondary color, links, magical accents | Mystical purple — conveys creativity, mystery, and arcane energy. |
| **Moonlight Cream** | `#FAF3E0` | Main background or neutral surface color | Off-white parchment tone — softens contrast, feels inviting and aged. |
| **Nightshade Blue** | `#1E2749` | Text, contrast backgrounds, headers | Deep indigo — adds depth and nighttime ambiance; pairs elegantly with warm tones. |
| **Ash Gray** | `#B8B8B8` | Subtext, icons, inactive states | Neutral gray — balances the palette, used for subtle UI hints and disabled items. |
| **Ember Red** | `#B33939` | Alerts, warnings, rare highlights | Muted red — feels organic, like embers or potion flames, not harsh. |
